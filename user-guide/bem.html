We started using BEM on the IQ project as a standardized naming convention. BEM is pretty simple to understand it stands for Block, Element, Modifier. Here's a simple explainer: https://seesparkbox.com/foundry/bem_by_example and another one: https://css-tricks.com/bem-101/ See this Confluence document for an explanation of how we're using BEM in IQ: https://docs.google.com/document/d/1F3T-avAqkZFup8kdppgFKVXBtLTzZypfrrBNTdZU6B4/edit#heading=h.v7xxkbyxm0a1
