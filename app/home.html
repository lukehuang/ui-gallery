Welcome to the Component Gallery

How to add a new page to the gallery

* create an html page using the following structure (see template.html) ```html <item-description> Usage docs <marked> You can use markdown inside <marked> tag </marked> </item-description> <item-example-html> example html </item-example-html> ``` * map the page name to the html page created above (add an entry to `config.js`) ```js 'Page name' : 'styles/page-name.html', ```
