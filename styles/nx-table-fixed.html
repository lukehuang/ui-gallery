Table with a fixed header and contents that scroll. This is a complex table that has many moving parts. - a parent `.nx-tile--table-name` with `positon: relative;` - a parent `nx-scrollable--table-name` which uses both the detect-scrollbar and maximize-container-height directives - `@include table-fixed-widths($numColumns)` in the custom `nx-scrollable--table-name` SCSS which has the table's column count as its input - in this case our table has five columns so our mixin would be set like this: `@include table-fixed-widths(5);` The mixin will work for simple tables which do not require fixed sizes for any of the table columns. For more complicated tables look at `_dashboard.scss` in the IQ Brain project for examples of tables with normal, expanding, and fixed width columns.
