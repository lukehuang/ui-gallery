`.iq-tile` is the base building block of our pages. There are three default classes that can be used within an `.iq-tile` `.iq-tile-header` is used for tile titles, it has title and sub-title elements (optional) `.iq-tile-content` for the tile content `.iq-tile-footer` for footer contents (buttons for example) `.iq-tile` can also be paired with `.iq-alert` styles to create tiles with alert coloring.
