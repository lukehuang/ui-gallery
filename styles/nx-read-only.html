`.nx-read-only` is used in instances when we want to display data or the output of a form in a read-only fashion. Typically this data takes the form of a label and the output. The output can have icons, and there can also be multiple outputs displayed horizontally. There are two different ways of coding nx-read-only to accomodate the ability to toggle within a form between a read only and an editable state. It is expected that the `<dl>` layout will be used in the vast majority of cases.
